# NOUS Protocol

This repository contains the NOUS protocol documentation and working specs.

## Contents
- `whitepaper/` — Whitepapers (FULL + spec)
- `docs/` — Investor memo and supporting docs
- `spec/` — Protocol specification drafts (to be expanded)
- `src/` — Reference implementations (placeholder)

## Quick start
This repo is documentation-first. If you're contributing code, start with:
1. Read the latest whitepaper in `whitepaper/`
2. Review `spec/` for normative requirements (coming next)
3. Open a proposal issue describing scope + threat model
